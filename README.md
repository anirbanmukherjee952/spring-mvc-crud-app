# spring-mvc-crud-app

**CRM (Customer Relationship Management):** A simple CRUD Application developed by using Spring MVC, Hibernate, JSP & JSTL. Following features are implemented:
1. All CRUD facilites.
2. Filter using search and sort.
3. Pagination.
4. Form validation.
